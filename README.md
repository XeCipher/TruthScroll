# TruthScroll

> The truth, one scroll at a time.

TruthScroll is an autonomous AI Agent that combats misinformation by hijacking the addictive "infinite scroll" format. It scans the web, verifies viral claims in real-time, and presents them as verified "Truth Cards."

**Submission for MumbaiHacks 2025 (Misinformation Track).**

---

### **Repository Structure**

- **[frontend/](./frontend)**: React + Vite application (Mobile & Desktop Responsive).
- **[backend/](./backend)**: Python Agent, Gemini Logic, and Reddit Ingestion.

### **Core Features**

- **The Truth Feed:** Instagram reel-style vertical scroll of verified facts.
- **Agent Console:** Real-time visualization of the AI scanning and verifying claims.
- **WhatsApp Buster:** Users upload screenshots; Agent uses Multimodal Vision to verify text.
- **Origin Graph:** Forensic visualization of a rumor's timeline (Origin -> Spread -> Verdict).
